  # Casos de prueba
  
  ### Prueba exitosa en /topsecret/
  
``` JSON
{
	"satellites": [
			{
			"name": "kenobi",
			"distance": 5.0,
			"message": ["este", "", "", "mensaje", ""]
			},
			{
			"name": "skywalker",
			"distance": 5.0,
			"message": ["", "es", "", "", "secreto"]
			},
			{
			"name": "sato",
			"distance": 13.0,
			"message": ["", "", "un", "", ""]
			}
	]
}
```
  Respuesta esperada:
``` JSON
  RESPONSE CODE: 200
{
    "message": "este es un mensaje secreto",
    "position": {
        "x": 1.0,
        "y": 1.0
    }
}
```
